# PackIntel Web

AI-driven EUC Operations Platform - Frontend

## Features

- Overview Dashboard with KPIs and alerts
- Vulnerability Intelligence Center (NVD/CISA integration)
- Application Management
- Workflow Studio (React Flow)
- Change Automation
- Communications Orchestrator
- Infrastructure Monitor
- AI Assistant (MCP-powered)
- Administration (Users, Integrations, Audit)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Zustand (State Management)
- TanStack Query (Data Fetching)
- React Flow (Workflow Canvas)
- Recharts & Tremor (Charts)
- Tailwind CSS
- Radix UI (shadcn/ui components)

## Environment Variables

```
NEXT_PUBLIC_API_URL=http://localhost:5000
```
